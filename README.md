# 开启服务

```
git clone git@github.com:jakequc/cross-domain-node-server-demo.git

npm i

# 配置你的 nginx

pnm run server

```

## next step

`yarn start`
or
`npm run start`
yet
`pnpm run start`

访问 [http://127.0.0.1:3000/](http://127.0.0.1:3000/)
